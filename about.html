About This App and the Developer

This app was created in 2021 by Colorado-based developer, Jason Pottorff, in the Career Foundry Web-Developer school program. The object was to create a simple, mobile-based application using HTML, CSS, Javascript, Bootstrap, JQuery, and a Pokemon API (hosted by CF). If you have any questions or would like to contact Jason, please go to his portfolio website.
